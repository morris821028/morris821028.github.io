Problem

在賽車運動中，常用 Lap 表示套一圈賽道所消耗的時間，現在有多名賽車手，給予最快一圈多久、最慢一圈多久，請問在最快車手第幾圈的時候，可以倒追最慢的車手。

Sample Input

Sample Output

Solution

懶得推導公式，二分倒追的時間，找到何時會產生倒追，再計算第幾圈即可。
