Problem

給定一個 N * M 的地圖，接著將會從起點出發抵達終點，一開始的速度為 (0, 0)，在下一個時刻，每一個維度的速度可以調整 -3, -2, -1, 0, 1, 2, 3 (也就是加速度)，但是速度的絕對值必須小於等於 3，同時給定這張地圖的矩形障礙物，移動的過程中不用考慮會不會遇到障礙物，只需要考慮跳躍落下的地點是否有障礙物。
