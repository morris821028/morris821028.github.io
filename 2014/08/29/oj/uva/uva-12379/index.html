Problem

給定一個樹狀圖 (題目說明任兩點只有一條唯一路徑)，打算選定一點作為送貨中心，貨車將從該地點出發，經過所有的點，目標最小化送貨時間。不用考慮最後停在哪裡，只需要考慮最後一個送達的時刻。

Sample Input

Sample Output

Solution

簡單地發現到，會將每條邊都經過兩次，而停留位置到出發位置的邊上只會經過一次，也就是說扣除掉最長路徑就是答案。(將最長路徑的其中一個端點當作送貨中心。)
