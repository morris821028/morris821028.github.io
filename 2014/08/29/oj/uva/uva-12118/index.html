Problem

將一個無向圖中，增加最少條邊，使得可以使用尤拉路徑或環道經過所有條邊。

Sample Input

Sample Output

Solution

同一個連通元件中，度數為奇數的點一定為偶數個，保留一組(兩個)奇數點，準備於其他連通元件串起，而其他的則兩兩相接(增加新的邊)，保證每一個點一定能構成偶數度。
