Problem

有一個半徑為 R 的圓草地，接著將一頭牛綁在圓周上 (定點)，請問繩長多少的時候，恰好能夠將圓佔有 P% 的面積。

Sample input

Sample output

Solution

將圓放在坐標軸上，假設綁定的位置於 (R, 0)，而接著二分另外一點 (於圓上) 可以拉到綁定的位置。

計算繩長和面積即可。
