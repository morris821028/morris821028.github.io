Problem

一個山峰高度 h[i] 如果算是突起，則必須符合

  • 到其他更高的山峰之間有經過低於 h[i] - 150000 的可能
  • 如果本身就是最高峰，則高度至少要 150000

Sample Input

Sample Output

Solution

維護一個遞增的單調堆，而第二個元素存放掃描時經過的最低點。然後從左往右掃一次、再從右往左掃一次，分別得到每一個點往左、往右找到最高峰的情況。
