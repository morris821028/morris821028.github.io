Problem

軍隊筆直等速度移動，隊伍長度 L，信使位於隊伍最後面，要傳遞訊息到隊伍前方後跑回隊伍末端。跑回末端時候，隊伍已經前進 L 距離。求信使跑的距離為何？

Sample Input

Sample Output

Solution

假設軍隊移動速度$v$, 信使移動速度$u$。

看時間

$$\frac{L}{v} = \frac{L}{u-v} + \frac{L}{u+v} \\ u^{2} - v^{2} = 2v, \\ \text{Let u = 1, get } v = \frac{-2 + \sqrt{8} }{2}$$

輸出$\frac{L}{v} u$ 即可。
