Problem

有五個人要測體重，倆倆上去秤得到體重總和為何，現在反推每個人的體重為多少。

Sample Input

Sample Output

Solution

這題跟 10202 - Pairsumonious Numbers 一樣。

如果將體重、總和排序後 A[]、SUM[]，保證最小的 A[0] + A[1] = SUM[0]，和 A[0] + A[2] = SUM[1] 但是不曉得 A[0] + A[3] 和 A[1] + A[2] 哪個小，因此窮舉 A[1] + A[2] 的結果。

每一次窮舉，可以解出前三小的值，之後就能依序推出所有結果。
