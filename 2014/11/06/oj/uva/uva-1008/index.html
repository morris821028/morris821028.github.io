Problem

給予一個 Vexed! 遊戲，這遊戲類似於消球遊戲，可以左右拖動特定元素使其掉落，在一個瞬間中可以讓相鄰兩個相同元素同時消失。在下一個瞬間，仍在盤面上的元素會根據重力往下墜落，如果又發現存有相同元素相鄰，則會不斷地迭代消去。

請問如何在最少步數下把所有元素消去。

Sample Input

Sample Output

Solution

對於每一個盤面做紀錄，並且對於每一個元素嘗試是否能向左向右拉動，拉動之後將其模擬墜落相消的過程。使用 bfs 的方式進行路徑搜索，隨後回溯即可。
