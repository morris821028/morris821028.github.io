Problem

機器人能轉移只有上下左右四個方向，機器人切換快速移動模式，將自己跨過小於等於 K 個連續的障礙物。也就是說，當障礙物連續出現形成一座厚牆時，機器人就爬不過去。

求最少移動次數。

Sample Input

Sample Output

Solution

將每一點多一個狀態 s，表示當前經過連續 s 個障礙物。如果下一個轉移地點不是障礙物，則 s 會歸零。
