Problem

給一個整數 N，接著詢問 x，有多少 gcd(i, N) <= x。

Sample Input

Sample Output

Solution

先把 N 的因數 f 全部找出，原則上不超過 2000 個，然後建出 phi(N/f[i])。

當要知道 gcd(i, N) = x 時，相當於找到 gcd(i/x, N/x) = 1，任何與 N/x 互質的個數。建造完後，對於每次詢問二分答案即可。

很多地方忘記用 long long，吃了不少 Runtime error。
