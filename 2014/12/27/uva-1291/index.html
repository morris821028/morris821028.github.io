Problem

跳舞機共有四個方向，兩隻腳必須在某一個時刻按到該按鈕。

一開始雙腳站立於 0 的位置，每一個時刻只能移動一隻腳。

每一次移動的花費，0 到任何位置都是消耗體力 2、其餘數字移動到相鄰格子消耗體力 3、其餘數字移動到相面格子消耗體力 4，腳不動消耗體力 1。

過程中雙腳不能站立於同一個格子。

Sample Input

Sample Output

Solution

dp[i][j][k] 分別記錄時刻 i，左腳所在位置 j，右腳所在位置 k。分別進行轉移即可。
