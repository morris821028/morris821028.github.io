Problem

給定三個曲線，每個曲線通過 n 個點，分別表示 r, g, b 的轉換圖示，請將影像的 rgb 替換成曲線給定的結果。

特別注意到曲線的最左、最右端點之外都是水平線。

Sample Output

Solution

處理最煩的就是通過 n 個點的曲線怎麼求，套用高斯消去法，假設多項式有 n 個係數求解。
