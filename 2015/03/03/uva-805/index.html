Problem

給兩個簡單多邊形，求交集的每一個簡單多邊形。

並且輸出時把共線的點移除，順逆時針都可以。

Sample Input

Sample Output

Solution

求兩個簡單多邊形的交集，用 double connected edge list 來解決交集好像是可行的操作？但是重疊在一起好像就會炸掉。

因此直接定向搜索 (將多邊形都以順或逆時針表示，接著產生可行的簡單多邊形) 任何可行簡單多邊形的存在判定！需要判定簡單多邊形的邊上中點是否都在兩個簡單多邊形內部，套用射線法的算法。
