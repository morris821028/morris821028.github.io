Problem

建造網路需要一個最少花費將所有點連起，然而有一個特別的方案，使得某些點集相連會有一個特殊的花費，這些特別的方案總數 < 8。

其餘任兩點需要相連，則將會花費兩點距離的歐幾里德距離平方，求一個最少花費。

Sample Input

Sample Output

Solution

看到 2D 平面，花費又是歐幾里得距離，可以利用 Delaunay 找到所有 EMST 的邊，邊的數量是線性 O(n) 的。

窮舉組合方案 O(2^8)，接著套用 MST 的算法來完成。
