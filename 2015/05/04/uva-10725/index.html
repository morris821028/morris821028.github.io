Problem

給三角形的三邊，放置一個最大面積的正方形於三角形中。

求最大面積為何。

Input

Output

Solution

把三角形的最大邊找到，考慮是鈍角還是銳角三角形，接著把正方形的某一邊貼在三角形上，用幾何計算三角形的三點座標，拉 45 度分角線得到正方形另一點座標，計算正方形邊長。
